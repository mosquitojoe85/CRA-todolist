CRA-todolist
===

嘗試以CRA製作一個todolist。
新增待辦事項，並可以修改事項完成狀態。

### 運行

輸入指令，
```
npm start
```
查看`http://localhost:3000/`即可。
因為牛牛很可愛，所以不支援IE瀏覽器。

### 額外安裝套件

- `cowsay`: 可愛的牛牛
- `qs`: url params處理
- `redux`
- `whatwg-fetch`: window.fetch的polyfill
- `node-sass`、`sass-loader`: SASS webpack loader
- `classnames`: 便於管理className
